Interview Questions

1. Difference between <div> and Semantic Tags?

<div> is a generic container without meaning. Semantic tags like <header>, <section>, and <footer> clearly describe the purpose of the content.

2. What is the CSS Box Model?

The CSS Box Model defines the spacing and sizing of HTML elements using:

  • Content
  • Padding
  • Border
  • Margin
